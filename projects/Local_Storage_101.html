Local storage ဆိုတာဘာလဲ
Local storage ဆိုတာနားလည်လွယ်‌အောင်ရှင်းပြရရင် data တွေကို key and value pairs ဖြင့်သိမ်းထားလို့ရတဲ့နေရာပဲဖြစ်ပါတယ်။ သူက browser က providede လုပ်တဲ့ feature တွေထဲကတစ်ခုဖြစ်ပါတယ်။ ပြောချင်တာကတော့ front end (client app) မှာ data တွေကိုသိမ်းလို့ရနိုင်တာပေါ့။

Common Uses
Saving user preferences: user ရွေးထားတဲ့ dark mode တို့ language တို့ကို local storage မှာပဲသိမ်းပါတယ်။
User data: JWT token လိုမျိုးတွေသုံးတဲ့ website တွေမှာဆိုရင် token ကို local storage ထဲမှာသိမ်းထားရပါတယ်။ အဲထဲကနေမှတစ်ဆင့်လိုအပ်တဲ့ user data တွေကိုထုတ်ယူပြီး authorization အတွက်သုံးကြပါတယ်။ သတိထားစရာဆိုရင် local storage က encrypted တာမဟုတ်တဲ့အတွက် Sensitive data တွေကိုသိမ်းထားခြင်းတော့ရှောင်သင့်ပါတယ်။ Storage limit လဲရှိတဲ့အတွက်လိုသလောက်ပဲသိမ်းတာမျိုးပဲလုပ်သင့်ပါတယ်။

Key Features of Local Storage
Local storage ထဲကရှိပြီသား data တွေကို ထုတ်နိုင်၊ဖျက်နိုင်တဲ့အပြင် front end (client app) မှာပဲလည်း data အသစ်တွေကို local storage ထဲကို ပါထည့်နိုင်ပါသေးတယ်။ ဒါ့အပြင် Persistent data: local storage ထဲက data တွေက browser ကိုပိတ်လိုက်ရင်တော့ကျန်သေးတဲ့အတွက် Persistent ဖြစ်တဲ့ထဲပါပါ‌ေသးတယ်။

Key-Value Pairs: local storage ထဲမှာ data တွေကို key and value pairs အနေနဲ့သိမ်းလို့ရတဲ့အတွက် manage လုပ်ရတာလွယ်ကူစေပါတယ်။ ဒါကတော့ front end development မှာအရေးကြီးပေမယ်လူသိပ်မပြောကြတဲ့ local storage အကြောင်းပါ။ ဒီ page မှာ programming နဲ့ပတ်သက်တာတွေတင်ပေးသွားမှာဖြစ်တဲ့အတွက်ကြိုက်နှစ်သက်ရင်တော့ like and follow လုပ်သွားလိုရပါတယ်။ ဒီ post ကိုကြိုက်နှစ်သက်ရင်လဲ like and share သွားလိုရပါတယ်။ ဒီနေ့တော့ဒီလောက်ပါပဲ။
