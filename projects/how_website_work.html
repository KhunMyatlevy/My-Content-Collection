Website တစ်ခုဘယ်လိုအလုပ်လုပ်လဲ

အားလုံးပဲ mingalar pr Hacker အေးမောင်ပြန်လာပါပြီ။ သင်အနေနဲ့ website တစ်ခုဘယ်လိုအလုပ်လုပ်လဲတွေးဖူးပါသလား။ ဒီနေ့မှာတော့ Website တွေဘယ်လိုအလုပ်လုပ်လဲဆိုတာကိုနားလည်လွယ်‌အောင်ရှင်းပြပေးသွားမှာပဲဖြစ်ပါတယ်။ ပထမဆုံးအနေနဲ့ website တစ်ခုဘယ်လိုအလုပ်လုပ်လဲသိရဖို့ website တွေရဲ့တည်ဆောက်ပုံကိုသိထားဖို့လိုအပ်ပါတယ်။ Website တစ်ခုကိုကျနော်တို့အနေနဲ့ frontend and backend ဆိုပြီးခွဲခြားလို့ရပါတယ်။ Frontend : User interface or user မြင်ရတဲ့အပိုင်းလိုအလွယ်မှတ်နိုင်ပါတယ်။ကျနော်တို့သုံးနေတဲ့ app တွေအကုန်မှာမြင်ရတဲ့ design လှလှလေးတွေက frontend အပိုင်းမှာပါဝင်ပါတယ်။ သူကို HTML, CSS and JavaScript ကိုသုံးပြီးရေးသားပါတယ်။ Backend : သူကတော့ website တစ်ခုရဲ့ logic တွေ function တွေကိုကိုင်တွယ်ရတဲ့အပိုင်းပါ။ ဥပမာပေးရရင် facebook လို app မျိုးမှာ search bar သုံးပြီးရှာရင်ကိုယ်ရှာချင်တဲ့ keyword ပါတဲ့ data တွေပြန်ပေးတာမျိုးပါ။ရှင်းရှင်းပြောရရင် website တစ်ခုရဲ့ function or feature တွေက backend အပိုင်းမှာပါဝင်ပါတယ်။ The Flow Frontend နှင့် Backend ကိုအနည်းငယ်ရှင်းပြပြီးရင်တော့ example နှင့်တူတူ website တစ်ခုဘယ်လိုအလုပ်လုပ်လဲပြောပြပါမယ်။ သင်က website တစ်ခုမှာ register လုပ်ဖို့အတွက်လိုလာပြီဆိုပါတော့။ အဲအခါမှာအဲ website ရဲ့ register လုပ်တဲ့နေရာကိုသွားပြီးလိုအပ်တဲ့ data တွေကိုဖြည့်ရပါတယ်။ အဲမှာမြင်ရတဲ့ form လှလှလေးတွေ register လုပ်ရင် click တဲ့ ခလုတ်လှလှလေးတွေက frontend ရဲ့အစိတ်ပိုင်းတွေပဲဖြစ်ပါတယ်။ ကိုယ်က register ဖို့လိုအပ်တဲ့အချက်လက်တွေ email and password လိုမျိုးတွေထည့်ပြီးရင်တော့ resigster or submite လိုခလုတ်မျိုးကို click ရပါတယ်။ ဒီအခါမှာ frontend ကနေ function တစ်ခုစ run ပါပြီ။ အဲ function ကတော့ frontend ကနေ backend ကို (request လုပ်တာလို့ခေါ်) သွားပြီ မောင်ဘယ်သူကတော့ register လုပ်ချင်ပါတယ်လို့သွားပြောပါတယ်။ backend က frontend ကလာတဲ့ request ကိုစစ်ဆေးပြီး resigster ပေးလုပ်ဖို့အဆင်ပြေတယ်ဆိုရင်အဲ user ပေးထားတဲ့အချက်လက်တွေနဲ့ User အသစ်တစ်ယောက်ကိုဖန်တီးပေးပါတယ်။အဲနောက်မှာတော့ resigster လုပ်တာအောင်မြင်ကျောင်းကိုပြန်ပြီး (respond) frontend ကိုပြောပါတယ်။ frontend ကအဲ backend ကလာတဲ့ respond ကိုကြည့်ပြီး user အမြင်လှအောင် user interface ကိုအဲပြန်လာတဲ့ respond အတိုင်းပြင်ပေးပါတယ်။
