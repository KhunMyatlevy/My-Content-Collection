SQL VS NO SQL

SQL vs NO SQL အားလုံးပဲ mingalar pr Hacker အေးမောင်ပြန်လာပါပြီ။ဒီနေ့မှာတော့ web development လုပ်ရာတွင်သော်လည်ကောင်း၊ တစ်ခြား application develop လုပ်ရာတွင်တွေ့ရလေ့ရှိတဲ့ SQL(relational databases) နှင့် NO SQL(non-relational databases) အကြောင်းပြောပြသွားမှာဖြစ်ပါတယ်။သူတို့ကဘာတွေလဲ၊ ဘာတွေကွာခြားမှုရှိလဲစတာတွေကိုပြောပြသွားမှာဖြစ်ပါတယ်။ SQL ဆိုတာဘာလဲ ? SQL ဆိုတာ (relational databases) လို့လဲသိကြပြီး application တွေတည်ဆောက်ရာမှတော်တော်အသုံးပြုကြပါတယ်။ သူကိုသုံးပြီး data တွေကို data base ထဲမှာဘယ်လိုသိမ်းကြလဲဆိုရင် table တွေနဲ့ row and column တွေနဲ့သိမ်းပါတယ်။ရှင်းရှင်းပြောရရင် excel spread sheets or google spread sheets တွေလိုမျိုး ဇယားနဲ့သိမ်းတယ်ပေါ့။ သူမှာ structured query language (SQL) လို့ခေါ်တဲ့ language ကိုသုံးပြီး database နဲ့ interact လုပ်ပါတယ်။ SQL example database တွေကတော့ MySQL, PostgreSQL, SQLite တို့ပဲဖြစ်ကြပါတယ်။ NO SQL ဆိုတာကကော့ဘာလဲ ? NO SQL data base နဲ့ SQL data base ကွာခြားချက်ကတော့ NO SQL က ဇယားနဲ့မသုံးတာလို့ပဲပြောရပါမယ်။ဇယားပဲမသုံးတာမဟုတ်ပဲသူက data တွေကို data base ထဲမှာသိမ်းတဲ့ပုံသေပုံစံမျိုးမရှိပါဘူး။ data တွေကိုသုံးရာတွံင် flexible ဖြစ်တယ်ပေါ့။ NO SQL ရဲံ data တွေကိုသိမ်းရာတွင်အသုံးများတဲ့ပုံတွေကတော့ Document-based (e.g., MongoDB) Document-based က‌တော့ရှင်းပြမတတ်တော့လို့ sample code ပဲကြည့်လိုက်ပါနားလည်သွားလိမ့်မယ်။ { "user_id": 123, "name": "John Doe", "email": "john@example.com", "orders": [ { "order_id": 1, "product": "Laptop", "quantity": 1 }, { "order_id": 2, "product": "Mouse", "quantity": 2 } ] } Key-value (e.g., Redis) javascript မှာဆို object လိုမျိုး၊ python မှာဆို dict လိုမျိုး။ data တွေကိုသူတို့ရဲ့ value တွေကို key name နဲ့တွဲထားတာမျိုး။ Columnar (e.g., Cassandra) သူကကြ SQL လိုမျိုးဇယားနဲ့ပဲ။ ဒါပေမယ်ကွာခြားချက်ကတော့သူက cell တစ်ခုခုမှာထည့်စရာ value မရှိရင်ဘာမှမထည့်ပဲနေ့လို့ရ။ SQL မှာဆို null တို့ဘာတို့တစ်ခုခုထည့်ပေးရ။ Conclusion SQL သေချာ structured ထားတဲ့ data၊ consistent ဖြစ်တဲ့ dataတွေနဲ့ complex queries ဖိုလိုတဲ့အခါမှာသုံးသင့်ပါတယ်။ NoSQL ကိုတော့ flexibility လိုတဲ့အခါ၊ scalability လိုတဲ့အခါနဲ့ unstructured data ကိုပမာဏအများနဲ့ကိုင်တွယ်ရတဲ့အခါမှသုံးသင့်ပါတယ်။ ဒါကတော့ Database ၂မျိုးနဲ့သူတိုအကြောင်း၊ ဘာတွေကွာခြားလဲဆိုတာပါပဲ။ ဒီpage မှာ programming နဲ့ပတ်သက်တာတွေတင်ပေးသွားမှာဖြစ်တဲ့အတွက်ကြိုက်နှစ်သက်ရင်တော့ like and follow လုပ်သွားလိုရပါတယ်။ဒီ post ကိုကြိုက်နှစ်သက်ရင်လဲ like and share သွားလိုရပါတယ်။ဒီနေ့တော့ဒီလောက်ပါပဲ။
