Event Handling

အားလုံးပဲ mingalar pr Hacker အေးမောင်ပြန်လာပါပြီ။ ဒီနေ့မှာတော့ front end development မှာ JavaScript ကိုသုံးရာတွင်အရေးကြီးတဲ့ event and event listener အကြောင်းကိုပြောပြသွားမှာဖြစ်ပါတယ်။

Event ဆိုတာဘာလဲ?
Event ဆိုတာ JavaScript မှာ user ဘက်ကနေ web page ကို click လိုက်တာဖြစ်စေ၊ keyboard မှာစာရိုက်တာဖြစ်စေ၊ mouse ကိုရွှေလိုက်တဲ့အခါတွေကိုဆိုလိုတာပါ။ ဒီလိုအခြေနေကို browser ကသိနိုင်ပြီး event listener ကိုသုံးပြီး ဒီလိုအခြေနေတွေမှာ JavaScript function တွေထည့်လို့ရပါတယ်။

Event Listener ဆိုတာကဘာလဲ?
Event Listener ကတော့ရှင်းပါတယ်။ သူက event တွေကိုစောင့်ပြီး ထို event တစ်ခုဖြစ်တိုင်း တစ်ခုခုဖြစ်ချင်ရင်ထည့်နိုင်တဲ့ JavaScript function တစ်မျိုးပါပဲ။ သူတို့က JavaScript ရဲ့ built-in function လဲဖြစ်ပါတယ်။

Common Use Case
Event နှင့် Event Listener တွေကို website ကို interactive ဖြစ်အောင်လုပ်ချင်တာခါမျိုးမှသုံးကြပါတယ်။ Button လိုမျိုးတွေဖန်တီးတာမျိုးမှာသုံးပါတယ်။ ထို button ကိုနှိပ်လိုက်ရင် function တစ်ခုကို run တာမျိုးတွေလုပ်လို့ရပါတယ်။ Mouse ရွေသွားတဲ့နေရာမှာ animation တွေဖန်တီးတာမျိုးတွေ၊ hover effect မျိုးတွေက event နှင့် event listener ကိုသုံးထားတာတွေကြီးပါပဲ။
