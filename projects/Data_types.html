What Are Data Types?

Data Types ဆိုတာနားလည်လွယ်အောင်ပြောရရင် software တည်ဆောက်ရမှာသုံးတဲ့ data တွေကိုအမျိုးစားခွဲတာလို့အလွယ်မှတ်လို့ရပါတယ်။ data တွေမှာ number(int or floate) တွေလဲရှိသလို text(string)လဲရှိပါတယ်။ ဒါတွေကိုအမျိုးစားတစ်ခုထဲအနေနဲ့သုံးလို့မရပါဘူး။ ဥပမာ 1 and 2 ကိုပေါင်းလို့ရပေမယ် 1 and a ကိုတော့ပေါင်းလို့မရပါဘူး။ ဒါတွေကို data types မခွဲထားရင်တော်တော်ရှင်းရခက်မှာပါ။

Common Data Types

တွေ့ရအများဆုံး data types တွေကတော့ Primitive Data Types and Composite Data Types တွေပဲဖြစ်ပါတယ်။

Primitive Data Types

Primitive Data Types ကတော့နားလည်လွယ်အောင်ပြောရရင် value တစ်ခုထဲပါတဲ့ data type လို့အလွယ်မှတ်နိုင်ပါတယ်( age = 25 )။ Primitive Data Types ထဲမှာ int or Integer သူတို့က အပြည့်ကိန်း value တွေဖြစ်ပြီး (age = 25 # Integer)၊ Float သူတို့က ဒသမကိန်း value တွေဖြစ်ပြီး (price = 19.99 # Float)၊ str or string သူတို့က text value တွေဖြစ်ကြပါတယ်။ (name = "Paul")။ double quote or single quote ကြားမှာရေးထားတဲ့မှန်သမျှကို string အဖြစ်သတ်မှတ်ပြီး number value များပါအပါဖြစ်တယ်(name = "Paul_25")ကလည်း string data type ပဲဖြစ်ပါတယ်၊ နောက်တစ်ခုကတော့ bool or Booleans။ သူကတော့ true or false value ကိုပဲပြပေးပြီးထူးခြားပါတယ်၊ကျနော်ကိုယ်တိုင်လဲ programming စစသင်ချင်းမှာခေါင်းစားစေခဲ့တဲ့ data type ဆိုလဲမမှားဘူး။ သူကိုအသုံးပြုပုံကိုဥပမာနဲ့ရှင်းပြပါမယ်။ User တစ်ယောက်နှင့်တစ်ယောက်အပြန်လှန် message ပို့လို့ရတဲ့ app တစ်ခုမှာ message ကိုလက်ခံတဲ့ user ကထို message ကိုဖတ်ပြီးသားလားမဖတ်ပြီးသားလားကို data base ထဲမှာ မှတ်ချင်ရင် bool or booleans ကိုသုံးနိုင်ပါတယ်။( ဖတ်ပြီးရင် true and မဖတ်ရသေးရင် false)

Composite Data Types

Composite Data Types ကတော့နားလည်လွယ်အောင်ပြောရရင် value တစ်ခုထက်ပိုပါတယ် data type ပဲဖြစ်ပါတယ်။ သူက Primitive Data Types နဲ့ ဆန့်ကျင့်ဘက်ဖြစ်ပြီး Arrays နှင့် Objects လို data type တွေပါဝင်ပါတယ်။ Arrays ဆိုတာကတော့ လေးထောင့်ကွင်း [ ] ထဲမှာထည့်ရေးပြီး value တစ်ခုထက်ပိုပါပါတယ်(numbers = [1, 2, 3, 4])၊ Objects ဆိုတာကတော့ တွန့်ကွင်း { } ထဲမှာထည့်ရေးပြီး value တစ်ခုထက်ပိုပါပါတယ်၊ သူရဲ့ထူးခြားချက်ကတော့ key value pairs ဆိုတဲ့ format နဲ့လာတာပါပဲ။ နားလည်လွယ်အောင်ပြောရရင် value တိုင်းမှာ key တစ်ခုစီနှင့်တွဲပါတာပါ (user = {"name": "Paul", "age": 25})။ ဒီဥပမာမှာ "Paul" က value ဖြစ်ပြီး name က "Paul" ရဲ့ key ဖြစ်ပါတယ်။

Static vs. Dynamic Typing

Static Typing ဆိုတာကတော့ data variable ကို value တစ်ခုနှင့် assign ရာတွင် data type ကိုပါတစ်ခါတည်းရေးပြီးသတ်မှတ်‌ပေးရတာမျိုးပါ။ (int age = 25)

Dynamic Typing ဆိုတာကတော့ data variable ကို value တစ်ခုနှင့် assign ရာတွင် data type ကိုယ်ဘာသာရေးစရာမလိုပါဘူး။ (age = 25)

Type Conversion

Type Conversion ဆိုတာကတော့ မတူညီတဲ့ data types တွေကိုလိုအပ်ချက်အရလိုအပ်တဲ့ပုံစံရအောင်ပြောင်းလဲခြင်းလို့အလွယ်မှတ်နိုင်ပါတယ်။ Type Conversion ကတော့ကိုယ်သုံးတဲ့ language ပေါ်မှုတည်ပြီးလုပ်ရပုံလဲအနည်းငယ်ကွဲပြားမှုရှိပါတယ်။

number = 42
text = str(number) # Convert number to string
print(text) # Output: "42"

ဒါကတော့ python မှာ 42 (int) ကို "42" (str) ဖြစ်အောင်လုပ်တဲ့ ဥပမာလေးတစ်ခုပါ။

Why Are Data Types Important?

Memory Efficiency: မတူညီတဲ့ data types တွေကကွဲပြားတယ် memory amount ကိုအသုံးပြုကြပါတယ်။ ဥပမာပေပေးရရင် float data type တွေက int data type ထက်ပိုပြီး space ပိုလိုပါတယ်။

Error Prevention: str နှင့် int data types တွေကိုပေါင်းလို့မရတဲ့အတွက် data type အကြောင်းသိထားမယ်ဆိုရင်ဒီ error တွေကိုကြိုတင်ကာကွယ်နိုင်မှာပါ။

Clarity: Data type အကြောင်းသေချာနားလည်ချင်းက debug ရတာပိုလွယ်စေပြီး code ကိုပိုနားလည်လွယ်စေပါတယ်။
