What is WebSocket?
WebSocket ဆိုတာ နားလည်လွယ်အောင် ပြောရရင် ပုံမှန် HTTP request တွေမှာ server ကပို့တဲ့ data တွေကိုရဖို့ browser ကို refresh လုပ်ရပါတယ်။ ဒါဟာ နောက်ဆုံး update data ကိုရဖို့အတွက် ဖြစ်ပါတယ်။ WebSocket မှာတော့ browser နဲ့ server ကြားမှာ connection တစ်ခုရှိသွားတဲ့အတွက် refresh စရာမလိုပဲ server က data တစ်ခုပို့လိုက်တာနဲ့ အဲ data ကိုတန်းရပါတယ်။ ဒီနေရာမှာ browser ကလည်း data တွေကို server ကို ပို့နိုင်ပါတယ်။

How does WebSocket work?
ပထမဆုံးအနေနဲ့ browser က server ကို WebSocket connection ပြောင်းရန် သာမာန် HTTP request တစ်ခု အရင်ပို့ရပါတယ်။ Server က WebSocket connection လုပ်ဖို့ သဘောတူလိုက်ရင်တော့ WebSocket connection ရပါပြီ။ WebSocket connection ရတာနဲ့ browser နဲ့ server ကြား communication ကနှောင့်နှေးမှာမရှိပဲ real-time နဲ့ လုပ်လို့ရနေမှာပါ။

WebSocket ကို Chat application တွေ၊ Live updates တွေတင်ပေးရတဲ့ Game တွေ၊ Live stream application တွေအပြင် real-time data update လိုတဲ့ application တော်တော်များများမှာ အသုံးပြုကြပါတယ်။ ဒါကတော့ Telegram တို့လို Chat application တွေမှာ အသုံးပြုကြတဲ့ WebSocket အကြောင်းပါ။
