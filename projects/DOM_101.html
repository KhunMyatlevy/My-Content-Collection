(Document Object Model) Dom ဆိုတာနားလည်လွယ်အောင်ပြောရရင်တော့ webpage ရဲ့ blue print နဲ့တူပြီး browser (JavaScript) ကိုထို webpage နဲ့ဘယ်လို interact လုပ်ရမလဲကိုလမ်းပြနိုင်ပါတယ်။ ဥပမာပေးရရင်အကယ်၍ webpage ကသာအိမ်တစ်လုံးဖြစ်ခဲ့မယ်ဆိုရင်အိမ်ကိုဆောက်ထားတဲ့အုတ်ခဲတွေက Html ဖြစ်ပြီးသုတ်ဆေးတွေက Css ဖြစ်မှာပါ။ browser (JavaScript) ကတော့နေထိုင်သူဖြစ်ပြီး DOM ကတော့ထိုအိမ်ရဲ့အသေးစိတ် design ပဲဖြစ်ပါတယ်။ DOM ကနေတဲ့လူ browser (JavaScript) ကိုဘယ်အခန်းက(Html) ဘယ်နေရာမှာရှိပြီးဘယ်လိုပုံစံ(Css)ရှိလဲဆိုတာကိုပြောပြပေးနိုင်ပါတယ်။ဒါမှနေတဲ့လူကအဲအခန်းတွေကိုသွားလာနိုင်မှာဖြစ်ပါတယ်။ ရှင်းရှင်းပြောရရင် DOM က JavaScript ကို webpage ရဲ့ Html and CSS တွေနဲ့ဘယ်လို interact ရမလဲဆိုတာပြောပြပါတယ်။ သူက conceptအရbackendရဲ့ Model နဲ့တောင်သွားဆင်တူနေပါပါတယ်။ Why the DOM matter? Website တစ်ခုရဲ့ interact ဖြစ်မူကအခြေနေအများစုမှာ DOM ကြောင့်ပါပဲ။ ပြောချင်တာက DOM မပါရင် website ကို interact and dynamic ဖြစ်အောင်လုပ်ရတာခက်ပါတယ်။ Live search results လို text တွေကို dynamically update လုပ်တာမျိုးတွေ၊ dark theme ပြောင်းတာမျိုးတွေဆို DOM ကိုအသုံးပြုရပါတယ်။ How JavaScript interact with DOM ? Selection specific element: ကိုယ် webpage ရဲ့ html element တစ်ခုကိုလိုချင်ရင် document.querySelector or getElementById ကိုသုံးပြီးလိုချင်တဲ့ element ကိုယူပြီး JavaScript functionality ထည့်လို့ရပါတယ်။ Changing content: html element တစ်ခုကိုယူပြီးတော့လဲသူရဲ့ value ကိုကိုယ်လိုချင်တဲ့အတိုင်း dynamically ပြောင်းလဲလို့လည်းရပါတယ်။ element.innerHTML = "Hello, World!" Adding interactivity: ဒီတစ်ခုကတော့ event listener တွေပါထည့်ပြီး webpage ကို interactive ဖြစ်အောင်လုပ်တာပါ။ event and event listener အကြောင်းကိုရေးစေချင်ရင်အောက်က comments မှာပြောသွားလို့ရပါတယ်။ element.addEventListener("click", function() { alert("Clicked!"); }); DOM ဆိုတာကတော့ front end development ရဲ့ model လိုဖြစ်ပြီး JavaScript ကို html element တွေနဲ့ interact လုပ်နိုင်စေပါတယ်။ JavaScript က html element တွေကို interact လုပ်နိုင်စေခြင်းဖြင့် webpage က dynamic and interact ဖြစ်လာမှာပါ။ ဒါကတော့ front end development မှာအခြေခံလည်းကျသလိုအရေးလည်းကြီးတဲ့ DOM ခေါ် Document Object Model ပဲဖြစ်ပါတယ်။ဒီpage မှာ programming နဲ့ပတ်သက်တာတွေတင်ပေးသွားမှာဖြစ်တဲ့အတွက်ကြိုက်နှစ်သက်ရင်တော့ like and follow လုပ်သွားလိုရပါတယ်။ဒီ post ကိုကြိုက်နှစ်သက်ရင်လဲ like and share သွားလိုရပါတယ်။သိချင်တာရှိရင်လဲ comments or cb မှာလာမေးလို့ရပါတယ်။ ရေး‌ေစချင်တာမျိုးရှိရင်လဲ cb or comments မှာရေးသွားလို့ရပါတယ်။ဒီနေ့တော့ဒီလောက်ပါပဲ။
