Front End 101

အားလုံးပဲ mingalar pr Hacker အေးမောင်ပြန်လာပါပြီ။ သင်အနေနဲ့ app တစ်ခုခုကိုသုံးရင်ဒီ app ကဘယ်လိုအလုပ်လုပ်နေလဲဆိုတာကိုတွေးမိဖူးပါသလား။ ကျနော်ကတော့တွေးမိဖူးပါတယ်။ ဒါကြောင့်ပဲဒီနေ့မှာတော့ website တွေနှင့် app တွေမှာအရေးပါတဲ့ front end နှင့် back end အကြောင်းကိုရှင်းပြပေးပြီး သူတို့တွေဘယ်လိုတွဲပြီးအလုပ်လုပ်လဲဆိုတာကိုပြောပြပေးသွားမှာဖြစ်ပါတယ်။

What is front end?
Front end ဆိုတာနားလည်လွယ်အောင်ပြောရရင် App တွေ website တွေမှာအသုံးပြုတဲ့ user မြင်ရတဲ့ content တွေကိုပြသပေးတယ်လိုအလွယ်မှတ်လို့ရပါတယ်။ သူကဘယ်လိုအလုပ်လုပ်လဲဆိုရင်ပထမဆုံး User က request တစ်ခုအရင်လုပ်ရပါတယ်။ (YouTube search bar မှာ Python full course ဆိုပြီးရိုက်ရှာလိုက်တဲ့အခါ) အဲ request(data) ကိုလက်ခံပြီး backend ဆီပို့ပေးရပါတယ်။ Back end ကပြန်ပို့ပေးတဲ့ respond (data) (videos with key word python full course) ကိုကြ user ကိုပြန်ပြပေးရပါတယ်။ ဒီနေရာ data တွေကိုအလွယ်တကူပို့ရတာမဟုတ်ပဲ လိုအပ်တဲ့ function တွေထည့်ရေးပြီးလိုချင်တဲ့ data format တစ်ခုရသည်ထိ process လုပ်ရပါတယ်။ Front end မှာတော့ data ကိုကြည့်လို့ကောင်းအောင်များသောအားဖြင့်ပြင်ကြရပါတယ်။

Key technologies
Html: Html ဆိုတာကတော့ front end development ရဲ့အခြေခံလို့တောင်ခေါ်လို့ရပါတယ်။ နားလည်လွယ်‌အောင်ပြောရရင်အိမ်တစ်အိမ်ရဲ့ နံရံ or structure နဲ့ဆင်ပါတယ်။ User ကိုပြမယ်ဘယ် content ကိုမဆို html ကိုသုံးပြီး design ရတာဖြစ်ပါတယ်။

CSS: ရှိပြီးသား html ကိုပိုလှသွားအောင်လုပ်ပေးတာကတော့ css ပါပဲ။ သူကြအိမ်တစ်အိမ်ရဲ့နံရံသုတ်ဆေး တို့အလှဆင်ထားတဲ့ပစ္စည်းနဲ့ဆင်ပါတယ်။ ကျနော်တို့သုံးနေတဲ့ app တွေ website တွေအကုန်လုံးက css ကိုသုံးပြီး styling လုပ်ထားတာပါ။ ဥပမာပေးရရင်ဂဏိန်းပေါင်းစက်မှာ ၁,၂,၃ လိုနံပတ်တွေကိုဘေးတိုက်မြင်ရတာက css ကြောင့်ဖြစ်ပြီး css သာမပါရင်အပေါ်အောက်တွေပဲအကုန်မြင်ရမှာပါ။

JavaScript: သူကတော့ထူးခြားပါတယ်။ ဘာလို့ဆို JavaScript က programming language တစ်ခုဖြစ်နေလို့ပါ။ ဒီလိုဆိုမေးနိုင်တဲ့ programming လုပ်နေပြီး programming language သုံးတာများထူးဆန်းလားလို့။ ထူးဆန်းပါတယ်။ front end ဆိုတာ browser ကပဲကိုင်တွယ်တာဖြစ်ပြီး programming language ဆိုတာသူရဲ့ code တွေအလုပ်လုပ်ဖို့ server လိုပါတယ်။ ရှင်းရှင်းပြောရရင် programming language ကသူအလုပ်လုပ်ဖို့ computer တစ်လုံးမဖြစ်မနေကိုလိုပါတယ်။ JavaScript ရဲ့ထူးခြားချက်ကတော့သူက browser ပေါ်မှာ run တာဖြစ်တဲ့အတွက် သူကိုသုံးဖို့ server မလိုပါဘူး။ programming language တစ်ခုဖြစ်တဲ့အတွက် database နဲ့ communicate လုပ်လို့မရတာကလွယ်ရင် JavaScript ကိုသုံးပြီးအကုန်လုပ်လို့ရပါတယ်။ ဒါဟာ front end development ရဲ့အသက်လိုတောင်ပြောလို့ရပါတယ်။ အထူးတစ်ဖြစ် back end ကို API အနေနဲ့ရေးလာတဲ့အခါမှာ data တွေ fetch ဖို့ programming language တစ်ခုကတော့လိုကိုလိုပါတယ်။ ဒါကို JavaScript ကအလွယ်တကူလေးလုပ်ပေးနိုင်ပါတယ်။

Frame works: React and Angular လိုမျိုး frame works တွေကတော့ဖြစ် JavaScript နဲ့ရေးရတာဖြစ်ပြီး front end development ကိုပိုမိုလွယ်ကူစေပါတယ်။ Front end ဆိုတာ web developer တော်တော်များများစလေ့လာကြတဲ့နေရာလဲဖြစ်ပြီး user နဲ့တိုက်ရိုက် interact လုပ်ရတဲ့နေရာလဲဖြစ်တဲ့အတွက်အရေးကြီးပါတယ်။ app or website တစ်ခုမှာ front end ကောင်းပြီးဆိုတန်းကိုသိသာပါတယ်။ အဓိကက user ကတိုက်ရိုက်မှတ်ချက်ပေးလို့ရတဲ့နေရာဖြစ်‌‌နေလို့ဖြစ်ပါတယ်။
