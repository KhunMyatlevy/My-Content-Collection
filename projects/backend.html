What’s up! Tech Nerds! Hacker အေးမောင်ပြန်လာပါပြီ။ ပြီးခဲ့တဲ့ post မှာ front end အကြောင်းရေးပြီးပြီဆိုတော့ဒီနေမှာတော့ backend အကြောင်းရေးချင်ပါတယ်။ Backend ဆိုတာ web development သာမက moible app နဲ့တစ်ခြား development တွေမှာပါအရေးပါပါတယ်။ ကျနော်ကတော့သူကို "the invisible engine that makes it all work" လိုတင်စားချင်ပါတယ်။ App တစ်ခုရဲ့ဦးနှောက်ဆိုလဲမမှားပါဘူး။ What is backend ? Backned ဆိုတာနားလည်လွယ်အောင်ပြောရရင် front end ရဲ့တောင်းဆိုချက်အရလိုအပ်တဲ့ data ကို database ကနေယူပေးရတာမျိုးပါ။ဒီအခါက data က process ဖို့လို့လာရင်လဲ backend က process ပေးပါသေးတယ်။ သူအလုပ်လုပ်ပုံက front end က request တဲ့ data ကို database ကနေသွားယူမယ်၊လိုအပ်ရင် process လုပ်မယ်ပြီးရင်တော့ front end ကိုပြန်ပြီး respond မယ်ပေါ့။ Backend ဆိုတာစားသောက်ဆိုင်ကမီးဖို့ချောင်နဲ့တူပါတယ်။ front end (စာသုံးသူရဲ့) request (order) ကို database ထဲက data (ကုန်ကြမ်းအသားငါးတွေကို) လိုသလို process (ချက်ပြုတ်ပေးပြီး) ပြန်ပေးတာပါပဲ။ Modern web development မှာဆိုရင် backend ကို api အနေနဲ့ရေးလာပကြပါတယ်။ ဒါဟာလဲ backend တစ်ခုထဲကိုသုံးပြီးမတူညီတဲ့ front end app တွေရေးနိုင်လို့ပါပဲ( mobile app, web app, desktop app)။ Responsibilities of Backend Backend ရဲ့ main Responsibilities တွေကတော့ Data base ထဲ data တွေအထုတ်သွင်းလုပ်တာ၊ app ရဲ့လုံခြုံရေးနှင့် user authentication၊ Business logic တွေထည့်သွင်းတာနဲ့၊ Api တွေကိုထည့်သွင်းတာတို့ဖြစ်ပါတယ်။ Key Technologies Backend မှာမဖြစ်မနေကိုပါရတာတွေကတော့ programming language တွေပါပဲ။ Function နှင့် Logic တွေကိုထည့်သွင်းမှာဖြစ်တဲ့အတွက် programming language ကတော့လိုကိုလိုပါတယ်။ ဒါကဘာကိုလဲဆိုလိုသလဲဆိုရင် backend app တစ်ခုကို host ဖို့အတွက် server တစ်ခုလိုတယ်ဆိုတာပါပဲ။ Git hub တို့ဘာတို့တွေမှာ host လိုမရဘူးလိုဆိုလိုတာပါ။ အသုံးများတဲ့ programming language တွေကတော့ JavaScript (Node.js) ကို Express.js လို framework မျိုးနဲ့တွဲသုံးတာကြပါတယ်။ Python ဆိုလဲ Django နှင့် Flask တို့လို framework တွေနဲ့တွဲသုံးကြပါတယ်။ Java ဆိုလဲ Spring Boot လို framework နဲ့တွဲသုံးကြပြီး PHP မှာဆိုလဲ Laravel လို framework မျိုးနဲ့တွဲသုံးကြပါတယ်။ Backend ဆိုတာ application တိုင်းရဲ့အသက်သွေးကြော ကျောရိုးဖြစ်ပြီးတော်တော်ကိုအရေးပါတဲ့အစိတ်ပိုင်းတစ်ခုဆိုလဲမမှားပါဘူး။ page မှာ programming နဲ့ပတ်သက်တဲ့ content တွေတင်ပေးသွားမှာဖြစ်တဲ့အတွက်ကြိုက်နှစ်သက်ရင် like and follow လုပ်သွားလိုရပါတယ်။ Post ကိုလဲ ကြိုက်နှစ်သက်ရင် share သွားလိုရပါတယ်။ဒီနေ့တော့ဒီလောက်ပါပဲ။
