API (Application Programming Interface)

ဒီနေ့မှာတော့ web development အပြင်တစ်ခြား Mobile App, Desktop App တွေတည်ဆောက်ရာမှာပါအသုံးများတယ် API အကြောင်းကိုပြောပြချင်ပါတယ်။

What is an API? (API ဆိုတာဘာလဲ)
စာဂျပိုးလို့‌‌ပြောရရင်တော့ Application Programming Interface။ ကျနော်ကတော့စာဂျပိုးမဟုတ်တဲ့အတွက် API ကိုစားသောက်ဆိုင်မှာအလုပ်လုပ်တဲ့ waiter အနေနဲ့ဥပမာပေးဖြစ်တယ်။ စားသုံးသူ (client app or front end) ရဲ့လိုအပ်ချက် (request) ကိုမီးဖိုချောင် (backend or server) ကိုသွားပြောပြီး မီးဖိုချောင် (backend or server) ကပြန်ပေးတယ်ဟင်းပွဲ (respond) ကို စားသုံး (client app or front end) ထံပြန်ပို့ပေးရတယ် process တစ်ခုလုံးက API အလုပ်လုပ်ပုံနဲ့တူတူပါပဲ။

လက်တွေ့ဘဝအသုံးပြုနေတဲ့ Google Map, payment method တွေနောက်ဆုံးဖုန်းမှာပါတဲ့ weather app ကအစ API ကိုသုံးပြီးရေးထားတဲ့ app တွေပဲဖြစ်ပါတယ်။ တစ်ခုနဲ့တစ်ခုမလုပ်လုပ်ပုံခြင်းတော့မတူကြပါဘူး။ Google Map ဆိုကိုယ်ရဲ့ location ကိုသိဖို့ကိုဖုန်းကနေ ကိုယ်ရဲ့ location ကို Google ကနေ fetch တာဖြစ်ပြီး Weather App ကြတော့ weather data တွေကို ကျနော်တို့ device တွေက weather API မှ fetch တာဖြစ်ပါတယ်။

Pros and Cons
API ရဲ့အားသာချက်တွေကတော့
- Third party tool and app တွေသုံးတယ်အတွက်ကြောင့် ကိုယ် application တွေကိုမြန်မြန်ဆန်ဆန်ရေးနိုင်စေခြင်း၊
- မတူညီတဲ့ front end app တွေကို server တစ်ခုထဲကိုသုံးစေနိုင်ခြင်း (ဥပမာ: Mobile app, Desktop app, and even Web app)၊
- အချိန်ကုန်သက်သာခြင်းတွေဖြစ်ပါတယ်။
ဒါအပြင်ငွေကိုတစ်ခါထဲ‌ချေရတဲ့ payment system ပါတဲ့ app တွေမှာဆို API ကမရှိမဖြစ်ကိုလိုအပ်တာဖြစ်ပါတယ်။ ဒီအချက်တွေကြောင့်ပဲ API ကလက်ရှိ backend development world မှာ demand အများဆုံး skills ဖြစ်နေရတာလဲဖြစ်ပါတယ်။ တစ်ချိန်ထဲမှာပဲ API က third party app တွေနဲ့အလုပ်လုပ်ရတာဖြစ်တဲ့အတွက် လုံခြုံရေးအရသတိထားရတာလေးတွေရှိပါတယ်။ ဒါကိုတော့ application ရဲ့လုံခြုံရေးကိုသေချာရေးခြင်းဖြင့်ကာကွယ်နိုင်ပါတယ်။

Modern Approach
လက်ရှိနည်းပညာ company ကြီးတွေကတော့ API ကို backend မှာရေးပြီး mobile app, desktop app, and web app တွေထိ backend တစ်ခုပဲသုံးကြတာကိုတွေ့နိုင်ပါတယ်။

API ဆိုတာ modern application development မှာအရမ်းအရေးပါပြီးမဖြစ်မနေကိုသိထားရမယ် topic တစ်ခုဖြစ်တဲ့အတွက်‌ကြောင့်လေ့လာထားဖို့အကြံ‌ပေးလိုပါတယ်။ ဒီ page မှာနောက်လဲ programming နဲ့ပတ်သက်တယ် content တွေတင်ပေးသွားမှာဖြစ်တဲ့အတွက် like and share လုပ်ထားလို့ရပါတယ်။
