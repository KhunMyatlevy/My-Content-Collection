Static Website vs Dyanmic Website

What’s up! Tech Nerds! Hacker အေးမောင်ပြန်လာပါပြီ။ ဒီနေ့မှာတော့ static website ဆိုတာဘာလဲ dynamic website ဆိုတာဘာလဲဆိုတာရှင်းပြရင်းနဲ့ဒီ website တွေကြားမှာဘာကွာခြားမှုရှိလည်းဆိုတာကိုပြောပြပေးသွားမှာပဲဖြစ်ပါတယ်။ စျေးကွက်ထဲမှာများသောအားဖြင့် static website နဲ့ dynamic website ဆိုပြီး ၂မျိုးကိုအဓိကတွေ့ရလေ့ရှိပါတယ်။ ၂ခုကြားကကွာခြားချက်ကိုနားလည်လွယ်အောင်ပြောရရင်တော့ static website ဆိုတာ content တွေကအသေဖြစ်နေတာဖြစ်ပြီး dynamic website ဆိုတာ content တွေကအသေမဖြစ်နေတာပါပဲ။ဒါကိုပိုနားလည်လွယ်အောင်တစ်ခုချင်းစီသေချာရှင်းပြပါမယ်။

Static website
Static website ဆိုတာ အပေါ်မှာပြောခဲ့တဲ့အတိုင်း content တွေ user ကိုပြချင်တဲ့ data တွေကအသေဖြစ်နေတာပါပဲ။ ဒီလိုဖြစ်နေရတဲ့အကြောင်းအရင်းဟာလည်း user ကိုပြချင်တဲ့ content တွေကို html ထဲမှာပဲအသေရေးထားတဲ့အတွက်ကြောင့်ဖြစ်ပါတယ်။ ဒီလိုဆို content တွေကို update လို့ရလားဆိုရင်တော့ html file ကို edit ပြီးတော့ update နိုင်ပါတယ်။ ဒီလိုလုပ်ဖို့အတွက် html code တွေကိုတော့နားလည်ဖို့လိုအပ်ပါလိမ့်မယ်။ Static website တွေကိုများသောအားဖြင့် Html, Css and Javascript ကိုသုံးပြီးရေးလေ့ရှိပါတယ်။ Content တွေအကုန်လုံးက pre written ဖြစ်တဲ့အတွက် dynamic website တွေထက်တော့ပိုမြန်ဆန်ပါတယ်။ Static website အနေနဲ့ရေးလေရှိတဲ့ website အမျိုးစားတွေကတော့ Portfolio websites, informational pages, business sites တွေပဲဖြစ်ကြပါတယ်။

Dynamic Websites
Dynamic Websites တွေကတော့ static website တွေနဲ့ဆန့်ကျင့်ဘက်ပါပဲ။ တည်ဆောက်ရတာလဲပိုမိုခက်ခဲပေမယ့်အများကြီးလဲပိုပြီးadvanced ဖြစ်ပါတယ်။ သူက တော့ user ပါ website နဲ့ interact လုပ်နိုင်စေပါတယ်။ ဒါကိုမြင်သာအောင်ပြောရရင်တော့ static website မှာ content ကိုအသစ်တင်ဖို့ or update ဖို့ code file ထဲဝင်ပြင်ရပေမယ့် dynamic website မှာတော့ user ကပါလုပ်နိုင်ပါတယ်။ သူမှာ Html, Css and Javascript အပြင် server side language တစ်ခုပါထပ်တိုးပါဝင်လာပါတယ်။ ပြောပြချင်တာကတော့ backend ပါလာပြီပေါ့။ ဒီအတွက် database လဲပါဝင်လာတယ်။ Performance ပိုင်းအရတော့ data တွေကို process ဖို့အချိန်ယူရတဲ့အတွက် static website တွေထက်တော့အနည်းငယ်ပိုနည်းနိုင်ပါတယ်။ Dynamic website ဆိုရင်လူသိများတာတွေကတော့ Social media platforms တွေနဲ့ e-commerce sites တွေပဲဖြစ်ပါတယ်။ ရှင်းရှင်းမှတ်ထားရင်တော့ frontend ပဲသုံးပြီးရေးထားရင် static website လို့မှတ်ယူထားနိုင်ပပြီး full stack app တွေကိုတော့ dynamic website လို့အလွယ်တကူမှတ်ယူနိုင်ပါတယ်။

ကဲဒါကတော့ static website နဲ့ dynamic website ကြားကကွာခြားချက်လေးတွေကိုပြောပြပေးသွားတာပဲဖြစ်ပါတယ်။ ဒီ page မှာ programming နဲ့ပတ်သက်တဲ့ content တွေဆက်လက်ပြီးတင်ပေးသွားမှာဖြစ်တဲ့အတွက် page ကို like and follow လုပ်ပေးဖို့မမေ့ကြပါနဲ့နော်။ ဒီ post‌လေးကိုကြိုက်နှစ်သက်ရင် like and share လုပ်ပေးဖို့တောင်းဆိုချင်ပါတယ်။ Tech Nerds တွေသာယာတဲ့နေ့ရက်လေးတွေပိုင်ဆိုင်နိုင်ပါစေ။
